В базе данных MS SQL Server есть продукты(products) и категории(categories). 
Одному продукту может соответствовать много категорий, 
в одной категории может быть много продуктов. 
Напишите SQL запрос для выбора всех пар «Имя продукта – Имя категории». 
Если у продукта нет категорий, то его имя все равно должно выводиться.



select products.name, categories.name from products
left join 
categories
on product.name = categories.productName
